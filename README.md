# Assembly Problem Solving Repository

This repository contains various Assembly language problems categorized by topic. 
Each problem has its own .asm file, which you can check and run.
The repo contains general problems and others from ICPC Assiut sheet.


### **How to Use This Repository?**
```
1. Clone the repository:
   git clone https://github.com/omnia197/Assembly-Problem-Solving.git

2. Navigate to a problem folder:
   cd Assembly-Problem-Solving/Basics

3. Open the `.asm` file in your preferred editor and run it in an assembler like mars.
```
